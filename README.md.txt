# FitWalk – Smart Outdoor Fitness Assistant 🚶‍♂️🌐🗺️

FitWalk is a web-based fitness tracker that helps users monitor their walking routes, internet connectivity, and display their path on a live canvas.

## 🔍 Problem Statement
Many people walk or jog without tracking route, idleness, or network loss. FitWalk uses smart web APIs to solve this problem.

## ✨ Features
- 🗺️ Live GPS tracking using Geolocation API
- 🖌️ Route drawn live on canvas using Canvas API
- 🌐 Internet speed/status shown using Network Info API

## 🔧 Web APIs Used
- [Geolocation API](https://developer.mozilla.org/en-US/docs/Web/API/Geolocation_API)
- [Canvas API](https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API)
- [Network Information API](https://developer.mozilla.org/en-US/docs/Web/API/Network_Information_API)
